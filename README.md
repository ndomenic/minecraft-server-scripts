# minecraft-server-scripts

A collection of Bash scripts to manage a Minecraft server

## Start a server

`./start.sh .env-server`

## Stop a server

`./stop.sh .env-server`

## View a running server

`./display.sh .env-server`

To detach from the server's screen session `Ctrl + a + d`

## Get server status

`./status.sh .env-server`
